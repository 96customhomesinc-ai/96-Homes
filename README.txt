96 Homes Website

How to preview locally:
1. Install Node.js
2. Open this folder in VS Code
3. Run:
   npm install
   npm run dev

How to deploy:
1. Upload this folder to Vercel
2. Vercel will detect Vite automatically
3. Deploy
4. Add your custom domain 96homes.ca in Vercel settings

Notes:
- Your logo is already included as public/logo-new.png
- The contact form sends to admin@96homes.ca through FormSubmit
- Send one test submission after going live so FormSubmit can verify the email
